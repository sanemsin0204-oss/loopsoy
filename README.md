# Loopsoy

A static, GitHub Pages-ready crochet portfolio and affiliate resource built with HTML, CSS, and vanilla JavaScript.

## Publish with GitHub Pages

1. Create a GitHub repository named `loopsoy` and upload these files.
2. In **Settings → Pages**, choose **Deploy from a branch**, then select `main` and `/ (root)`.
3. Replace `YOUR-USERNAME` in `sitemap.xml` and `robots.txt` with your GitHub username.
4. Update the placeholder social links, contact email, and Amazon `#` links before publishing.

## Add a new project

Duplicate `projects/project-template.html`, rename it, replace its content and images, and add its final URL to `sitemap.xml`.

## Important

The blue puff bag photo in `assets/images/` is user-supplied. Only use photos and product assets you have permission to publish. Add real Amazon Associate links only after you have enrolled in the programme.
